first book！

先附上安装方法：

首先下载一个nodejs的稳定版本，不然可能出问题：

  • cb函数_如何在使用Gitbook时解决“ cb.apply不是函数”错误

  • TypeError ERR_INVALID_ARG_TYPE: The "data" argument must be of type string or an instance of Buffe

下载nodejs的smi文件后，右键安装。

在命令行输入：

npm install gitbook-cli -g

然后可以在指定文件夹创建：

gitbook init

文件夹中会出现README 同 SUMMARY两个文件。
